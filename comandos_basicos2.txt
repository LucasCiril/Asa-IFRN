Comandos básicos 2

1- Como criar uma rede?
docker network create <NAME>

2- Como listar as redes disponíveis?
docker network ls

3- Como adicionar um container a uma rede?
docker run [-dp <HOST_PORT:CONTAINER_PORT>] --net <NET_NAME> <IMAGE_NAME>

4- Como excluir uma rede?
docker network rm <NET_NAME>
